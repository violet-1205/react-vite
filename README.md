# React + Vite 프론트엔드

간단하고 빠른 개발 경험을 제공하는 React + Vite 템플릿입니다. HMR(Hot Module Replacement)과 기본 ESLint 규칙이 포함되어 있어 즉시 개발을 시작할 수 있습니다.

## 기술 스택
- React 19
- Vite 7
- ESLint 9
- @vitejs/plugin-react

## 시작하기
아래 명령어로 개발 서버를 실행할 수 있습니다.

```bash
npm install
npm run dev
```

빌드 및 미리보기:

```bash
npm run build
npm run preview
```

코드 검사:

```bash
npm run lint
```

## 프로젝트 구조
- `src/` 애플리케이션 소스 코드
- `index.html` 진입 HTML
- `public/` 정적 자산
- `vite.config.js` Vite 설정

## 기타
- React Compiler는 기본적으로 활성화되어 있지 않습니다. 필요 시 공식 문서를 참고하여 추가할 수 있습니다.
- TypeScript 사용을 원하면 React + TS 템플릿을 고려해 주세요.
